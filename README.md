# Grab-Like Booking Platform (All-in-One HTML)

## 🚗🛵 Car & Motorcycle Booking + Travel Planning App

This is a fully functional **car and motorcycle booking platform** inspired by **Grab**, built entirely in **a single HTML file** with inline CSS and JavaScript.

The platform combines modern travel planning with comprehensive trip management and a user-friendly UI designed to function similarly to Grab, but tailored for **personal travel** including **itineraries, bookings, and trip planning tools**.

---

## 🌟 Features

✅ **Three-Panel Layout (Responsive)**  
- **Header:** Search bar, trip selector, user profile with stats  
- **Left Sidebar:** Trip navigation, saved destinations, travel guides  
- **Main Content:** Interactive maps, itineraries, destination details  
- **Right Sidebar:** Trip summary, weather, recommendations  

✅ **Full Travel Functionality (Simulated Client-Side)**
- 🔍 Destination Search with Autocomplete  
- ✈️ Trip Creation + Management  
- 📆 Itinerary Builder with Drag-and-Drop Scheduling  
- 🏨 Booking Management (Flights, Hotels, Activities)  
- 💰 Budget Tracking & Visualizations  
- 📑 Travel Document Organizer with Reminders  
- 🌦️ Weather Integration with Animated Cards  
- 🗺️ Map with Points of Interest (POIs) and Route Planning  
- 📦 Packing List Generator  
- 📓 Travel Journal with Photo Upload Simulation  
- 📤 Trip Sharing & Collaboration Simulation  
- 📴 Offline Access (via `localStorage`)  

✅ **Design Highlights**
- Parallax destination cards  
- Animated weather widgets  
- Interactive map clustering (simulated)  
- Timeline view for itinerary  
- Floating Action Buttons (FABs)  
- Material Design Icons ([Google Fonts](https://fonts.google.com/icons))  
- Fully responsive:  
  - 🖥 Desktop: 3-panels  
  - 📱 Mobile: Single panel  
  - 📲 Tablet: Two panels  

---

## 🧱 Technologies Used

- **HTML5** (single file)
- **CSS3** (inline, fully responsive, modern design)
- **Vanilla JavaScript** (all logic, DOM interaction, drag-and-drop, simulation)
- **Google Fonts + Material Icons**
- **Placehold.co** for placeholder images

---

## 📁 File Structure

- `grab-like-booking-platform.html`  
  A single HTML file containing all code, layout, styles, and scripts.

> No external libraries or APIs are used. All functionality is simulated client-side.

---

## 🚀 How to Use

1. **Download or clone** the repository.
2. Open `grab-like-booking-platform.html` in any modern browser.
3. Interact with the app: create trips, build itineraries, manage bookings, view maps and recommendations.

---

## ⚠️ Limitations

- No backend/API integration — all data is temporary or stored via `localStorage`.
- Map and weather integrations use **mock data/placeholders** for offline simulation.
- File uploads and sharing are simulated only — no real server interaction.
